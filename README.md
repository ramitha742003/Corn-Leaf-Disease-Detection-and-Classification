# Corn Leaf Disease Detection and Classification using CNN and Image Processing

Designed and trained a CNN-based model capable of detecting and classifying multiple corn leaf diseases with high accuracy. 

Evaluated the systemâ€™s performance using appropriate metrics like confusion matrix, accuracy, precision, recall, etc.

Implemented a Streamlit web application for real-time disease detection from uploaded leaf images.

Tools used: Jupyter notebook, Python, Streamlit, sklearn, Tensorflow/keras, Matplotlib/Seaborn, numpy.

# Kaggle Dataset

kaggle dataset for corn leaf images - https://www.kaggle.com/datasets/abdelrahmanemad2199/corn-or-maize-leaf-disease-dataset

kaggle dataset for natural images (not corn leaf) images - https://www.kaggle.com/datasets/prasunroy/natural-images
